# HustleNest v3.0 Release Notes

**Release Date:** February 4, 2026

---

## 🎉 What's New in v3.0

This major release introduces powerful new features for customization, data management, and reporting.

### 🌙 Dark Mode
- Full dark theme support across the entire application
- Toggle between Light and Dark modes via **Settings → Advanced → Appearance**
- Persists across sessions

### 📊 Dashboard Customization
- **Collapsible Sections** – Click any dashboard group header to collapse/expand
- **Hide/Show Sections** – Control visibility of individual dashboard panels
- **Persistent State** – Your layout preferences are saved automatically
- Configure via **Settings → Advanced → Dashboard**
- Quick actions: Expand All, Collapse All, Show All, Reset to Defaults

### 📈 Multiple Chart Types
- **Bar Charts** – Traditional vertical bar visualization
- **Line Charts** – Trend analysis with connected data points
- **Pie Charts** – Proportional breakdown of sales data
- Switch chart types from the dashboard toolbar

### 🗑️ Soft Delete & Trash System
- Deleted items are moved to Trash instead of permanent deletion
- **30-day retention** before automatic permanent deletion
- Restore accidentally deleted orders with one click
- Access Trash via **View → Trash** or the toolbar

### 💾 Database Backup Scheduler
- Automatic scheduled backups (Daily, Weekly, Monthly)
- Configurable backup retention (keep last N backups)
- Manual backup trigger available
- Configure via **Settings → Advanced → Backups**

### 📥 Data Import Wizard
- Import orders from **Excel (.xlsx)** and **CSV** files
- Column mapping interface for flexible imports
- Preview data before committing
- Access via **File → Import Data**

### 📄 PDF Report Exports
- Export professional PDF reports for:
  - Sales summaries
  - Order details
  - Customer reports
  - Financial statements
- Customizable date ranges and filters

### ⏭️ Skip Version Feature
- Choose to skip specific update versions
- Won't be prompted again for skipped versions
- Can reset skipped versions in Settings

## 💻 System Requirements

- **OS:** Windows 10 or later (64-bit)
- **RAM:** 4 GB minimum, 8 GB recommended
- **Disk Space:** 500 MB for installation
- **Display:** 1280x720 minimum resolution

---

## 📦 Installation

1. Download `HustleNestSetup.exe`
2. Run the installer
3. Follow the installation wizard
4. Launch HustleNest from the Start Menu or Desktop shortcut

---

## 🔄 Upgrading from Previous Versions

- Your existing database will be automatically migrated
- Settings and preferences are preserved
- Backup your data before upgrading (recommended)
